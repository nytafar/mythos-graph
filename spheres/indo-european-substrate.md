# Indo-European substrate sphere

## Frame commitments

- **Cross-IE comparative method.** Vedic, Iranian, Greek, Roman,
  Germanic, Celtic, Baltic, Slavic — each substrate is read against
  the others for shared cosmological grammar. Method is
  philological-structural: what the languages preserve, what the
  myths share at deep structure, what ritual patterns recur.
- **Sky-god → storm-god transition.** The reconstructed
  Proto-Indo-European *Dyēus* (sky-father) survives in Vedic
  Dyaus, Greek Zeus, Roman Jupiter, Germanic Týr/Tiw,
  Baltic Dievas; in many substrates the sky-father becomes a
  *deus otiosus* and active cult passes to a storm/weather god —
  Indra, Thor, Taranis, Perkūnas, Perun
  ([[eliade-sacred-and-profane#^p166]]–[[eliade-sacred-and-profane#^p167]]).
- **Cosmic-tree complex.** Yggdrasil ([[eliade-sacred-and-profane#^p207]]),
  the Vedic *aśvattha* and *skambha*, the Achilpa pole at
  [[eliade-sacred-and-profane#^p044]]–[[eliade-sacred-and-profane#^p045]],
  the Iranian *gao-kərəna* — the cosmic-tree pattern is densely
  attested across IE substrates, with substrate-specific
  augmentations (Yggdrasil's gnawing the most divergent).
- **Tripartite social-cosmic schema.** Dumézil's *trois fonctions*
  pattern (sovereignty / warfare / fertility) recurs across IE
  pantheons: Mitra-Varuna-Indra-Aśvins, Jupiter-Mars-Quirinus,
  Óðinn-Þórr-Freyr/Freyja. Operative as comparative scaffold
  even where contested empirically.
- **Hierogamy at cosmogonic scale.** Sky-Father / Earth-Mother
  union as cosmogonic act — Dyaus/Pṛthivī, Ouranos/Gaia,
  Vedic groom-and-bride identifying as heaven-and-earth at
  [[eliade-sacred-and-profane#^p203]]. Marriage rites imitate
  the cosmic hierogamy.
- **Shared cosmological grammar without single origin.** The
  sphere does not assume a single Urvolk or pure Proto-IE
  cosmology. It assumes that *the substrates are deep-time
  related* — through migration, contact, and shared inheritance —
  and that the comparative reading surfaces structural patterns
  no single substrate carries alone.
- **Substrate-comparative discipline against universalism.**
  Distinct from eliadean-sacred (which is methodologically
  phenomenological and universal-claiming across all substrates),
  this sphere is comparative and substrate-bounded — claims hold
  across IE substrates, not necessarily across non-IE substrates.

## Validation moves

When pressure-testing canon against this sphere:

- For any cosmological claim the brand makes that draws on a
  single IE substrate (Norse, Greek, Christian-IE), check whether
  the claim is substrate-specific or substrate-shared. A claim
  presented as universal that holds only in one substrate has
  been overgeneralised.
- For sacred-figure claims, check the IE comparative literature for
  the figure's cognates. A "sky god" claim in Norwegian-substrate
  brand work that misses the Týr/Tiw → Óðinn displacement is
  missing the sphere's load-bearing diachronic story.
- For tree / pillar / mountain axis-mundi claims, distinguish the
  shared IE structure (axis as connecting form between cosmic
  zones) from substrate-specific augmentations (Yggdrasil's
  gnawing, the cosmic-mountain's monsoonal verticality).
- Check Dumézilian function-fit when the brand claims sovereignty,
  warrior, or fertility-grounded virtue; the tripartite frame
  illuminates which substrate-pattern the brand is operating in.

## Failure symptoms

Canon failing this sphere will:

- Read Norse cosmology as Norwegian-only, missing the IE comparative
  context that tells which features are substrate-specific
  divergence and which are shared inheritance
- Treat Christian sacramental cosmology as discontinuous with the IE
  substrate it grew through (the deus-otiosus pattern, the cosmic-
  tree-as-cross, the hierogamy in marriage rites all carry forward)
- Use "ancient wisdom" or "primordial pattern" language that
  collapses substrates without naming the substrate. The sphere
  preserves substrate-specificity inside the comparative move.
- Borrow figures from one IE substrate while applying frame from
  another (e.g., reading Thor in Hellenic register, or Yggdrasil
  in Mesopotamian register).

## Correctives

When failure fires:

- Locate the substrate the brand actually inhabits (for
  Norwegian-substrate brand work: Germanic-Norse with Christian
  overlay).
- Use the comparative IE move to *inform* substrate-specific work,
  not to *replace* it. Comparative reading shows where the
  substrate's pattern is shared and where it is divergent.
- Where the brand draws on multiple IE substrates, name the
  substrates and the relation, not "the West" or "European
  tradition" generically.

## What this sphere centers, touches, refuses

**Centers.** The cross-IE substrate-comparative frame: shared
cosmological grammar read across Vedic, Iranian, Greek, Roman,
Germanic, Celtic, Baltic, Slavic substrates with substrate-specific
augmentations preserved.

**Adjacent-touches.**

- [[eliadean-sacred]] — Eliade is adjacent to the sphere because he
  *draws comparatively* from IE traditions in *The Sacred and the
  Profane* without being substrate-comparative-disciplined. He
  treats IE substrates as instances of the universal hierophany
  pattern; the sphere treats IE substrates as a comparative set
  with its own structural integrity.
- [[Norwegian-folkloric]] — the Norwegian substrate is one IE
  substrate; this sphere is the broader comparative set that lets
  the substrate-vs-universal pressure-test work without forcing
  collapse. The Norwegian substrate's cosmic-tree (Yggdrasil) reads
  in IE comparative context as gnawed-tree augmentation, not as
  outlier from a generic "world-tree" universal.

**Distinct from.**

- [[Norwegian-folkloric]] — that sphere is substrate-specific (Norse
  / late-medieval Norwegian / folkway), this sphere is the broader
  comparative IE set
- [[eliadean-sacred]] — that sphere is methodological-phenomenological,
  this sphere is comparative-philological. The two can read the
  same passage and produce different operative findings.

## Norwegian-substrate touchpoint

The Norwegian substrate is one specific IE substrate, dense in
preserved material because of late Christianisation and the
folklore-collection energy of nineteenth-century Norway (Asbjørnsen,
Moe, Lie). The sphere's relation to the substrate has three modes:

1. **Substrate-as-instance.** Yggdrasil instances the IE
   cosmic-tree pattern; Týr instances the IE *Dyēus* deus-otiosus
   pattern displaced by Óðinn (storm/wisdom god) in the Germanic
   substrate; Þórr instances the IE storm-god active-cult pattern.
   The comparative reading lets Norwegian-substrate work draw on
   the broader pattern without flattening the substrate's own
   shape.
2. **Substrate-as-divergence.** Yggdrasil's gnawing
   ([[wiki/concepts/cosmic-tree]] flag), Ragnarǫk's irreversibility
   ([[wiki/concepts/eternal-return]] flag), the gard-utgard
   *locational* sacred-profane (the woods *are* utgard regardless
   of who walks them) — each is a Norwegian-substrate divergence
   from the IE shared pattern. The comparative frame surfaces these
   as substrate-specific, not as data points against a universal
   claim.
3. **Substrate-as-augmentation.** Norwegian substrate carries
   stewardship-against-hostile-forces as a cosmological dimension
   the broader IE pattern has at lower density. The cosmic-tree-
   as-stewarded-presence is a Norwegian-substrate augmentation
   that the IE comparative set can carry forward (the Iranian
   *gao-kərəna* tree gnawed by the lizard is the closest IE
   parallel — Iranian and Norse together suggest a substrate
   pattern Eliade's universal frame missed).

The sphere is the right home for substrate-vs-universal pressure-
test work — it is broad enough to admit substrate variations
without forcing collapse, and disciplined enough to preserve
substrate-specificity inside the comparative move.
